# HealthHealthABC Backend

This is the backend for the HealthHealthABC project. It is a Kotlin Spring Boot project that uses the Spring Data JPA
library to interact with a MySQL database. The project also has a Redis cache to store data such as email confirmation
and recovery codes for forgotten passwords.
